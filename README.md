# Cloud Computing and FinTech
## Course Description
課程內容包括財金專業知識、文字探勘與機器學習，以及使用程式語言（如Python）來實作專題題目。

學習 AWS 所提供的雲端運算服務，其中包含：
* 架設虛擬機與伺服器
* 網頁部署與版本控制
* API 串接 Chatbot
* RDS 與 DynamoDB 資料庫服務
* Docker 技術應用

除此之外，將會跨校組隊，共同解決金融業者（如：南山人壽、永豐金控等公司）所提出的實務問題。
